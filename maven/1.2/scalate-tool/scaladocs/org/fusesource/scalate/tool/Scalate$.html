org.fusesource.scalate.tool

Scalate

object Scalate extends AnyRef

The extensible command line tool for Scalate.

Add new commands by:

  • Implementing the the Command trait
  • Add the class names of the new commands to a META-INF/services/org.fusesoruce.scalate/commands file in your jar
  • Drop your new jar into the ${scalate.home}/lib directory.
